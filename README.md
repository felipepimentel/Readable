## Running

To get started developing right away:

* Clone Repository
	- `git clone https://github.com/felipepimentel/Readable.git`
	- `cd Readable`
	
* Install and start the API server
    - `cd api-server`
    - `npm install`
    - `node server`
	
* In another terminal window
    - `cd client`
    - `yarn`
    - `yarn start`
